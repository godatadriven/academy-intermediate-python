# Exercises

For each exercise create a new file with the name of the exercise (eg. `exercise-1A`) in the `exercises/` folder.

## Out of Jupyter

|Name of Challenge|Description|
|----|----|
|Exercise 1A|Count the number of times the word "the" appears in a text file.|
|Exercise 1B|Count the mean number of words per sentence for a certain text file.|
|Exercise 1C|Find the 10 most common words in a certain text file.|
|Exercise 1D|Count the number of times a chosen word appears in a text file. You should be able to parse both the word and text file as arguments.|
|           |BONUS: Include an option to make the count case sensitive.|


**How to do these exercises:** Use the code in the `existing-code/` folder. Do not rewrite the way the code functions, instead use what you have learnt about writing code in the IDE (using functions `if __name__='__main__'` etc.).
